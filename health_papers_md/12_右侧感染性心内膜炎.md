# 右侧感染性心内膜炎：近期流行病学变化

**作者：** Shi-Min Yuan（袁世民）
**单位：** 福建医科大学附属莆田市第一医院心胸外科
**来源：** Int J Clin Exp Med, 2014;7(1):199-218
**ISSN:** 1940-5901

---

## 摘要

**背景：** 感染性心内膜炎（IE）的报道日益增多，但关于右侧IE的近期发展信息较少。

**方法：** 基于近期5⅓年的文献对右侧IE进行综合分析。

**结果：** 入侵门户、植入异物和修补的先天性心脏缺损是主要的易感危险因素。右侧瓣膜上的赘生物大小明显小于瓣膜以外的赘生物。多元逻辑回归分析显示，易感危险因素以及赘生物大小和位置是患者生存的独立预测风险。

**结论：** 过去5⅓年右侧IE的变化包括：患者年龄更轻、赘生物增大，但金黄色葡萄球菌感染仍占主导地位。并发症谱已转变为更多瓣膜关闭不全、更多栓塞事件、脓肿形成减少、瓣膜穿孔显著减少。在有效的抗生素方案治疗下，患者预况似乎优于以往。

**关键词：** 抗菌药物，血源性病原体，心脏外科手术，并发症，栓塞

---

## 引言

感染性心内膜炎最常累及主动脉瓣，其次为二尖瓣，三尖瓣和肺动脉瓣最少见。多瓣膜受累见于17-22%的患者。右侧IE占所有IE的5-10%。在先天性心脏缺损患者中，左侧、右侧和双侧IE分别占46.4%、32.7%和2.3%。

值得注意的是，过去几十年IE的流行病学和预防策略发生了持续变化。留置导管、异物医疗器械植入和静脉吸毒者已成为细菌定植的日益增长的危险因素。金黄色葡萄球菌已成为IE最常见的微生物，而草绿色链球菌感染减少。

本研究旨在基于近期5⅓年文献，呈现右侧IE在当前抗菌治疗下的流行病学、易感危险因素、微生物学和预后的变化趋势。

---

## 材料与方法

在PubMed数据库和Google搜索引擎中检索2008年1月1日至2013年4月30日的5⅓年近期文献。搜索词包括"右心内膜炎"、"右侧心内膜炎"、"肺动脉瓣心内膜炎"、"三尖瓣心内膜炎"、"起搏器导线心内膜炎"等。

共获得168篇文献（14篇原始文章、8篇病例系列和146篇病例报告），包括299名患者。

---

## 结果

### 临床表现

| 临床表现 | 例数（%） |
|---------|----------|
| 发热 | 152（91.6%） |
| 乏力/疲劳/不适/虚弱 | 53（31.9%） |
| 呼吸困难 | 37（22.3%） |
| 咳嗽 | 29（17.5%） |
| 体重减轻 | 17（10.2%） |
| 胸痛 | 18（10.8%） |
| 盗汗 | 12（7.2%） |
| 充血性心力衰竭 | 10（6.0%） |
| 胸腔积液 | 12（7.2%） |
| 心包积液 | 9（5.4%） |
| 肝肿大 | 11（6.6%） |

### 患者特征

262名患者包括182名男性和80名女性，男女比2.3:1。患者年龄40.2±21.3岁（范围0.04-88岁，中位数41岁）。

### 易感危险因素

入侵门户、植入异物和修补的先天性心脏缺损是主要的易感危险因素。静脉吸毒是重要的危险因素之一。

### 微生物学

金黄色葡萄球菌是右侧IE最常见的病原体。草绿色链球菌和其他链球菌也是重要的致病菌。

### 赘生物特征

右侧瓣膜上的赘生物大小明显小于瓣膜以外的赘生物。赘生物位置和大小是患者生存的独立预测因素。

### 并发症

- 瓣膜关闭不全：增加
- 栓塞事件：增加
- 脓肿形成：减少
- 瓣膜穿孔：显著减少

### 治疗和预后

在有效的抗生素方案治疗下，结合必要的心脏外科手术，患者预后似乎优于以往。多元逻辑回归分析显示，易感危险因素以及赘生物大小和位置是患者生存的独立预测风险。

---

## 讨论

本研究基于5⅓年文献综合分析了右侧IE的流行病学变化。主要发现包括：

1. **患者年龄更轻：** 过去右侧IE主要见于年轻静脉吸毒者，现在年龄分布更广泛
2. **赘生物增大：** 可能与诊断技术改进和报告偏倚有关
3. **金黄色葡萄球菌仍占主导：** 与整体IE趋势一致
4. **并发症谱变化：** 更多瓣膜关闭不全和栓塞事件，但脓肿和穿孔减少
5. **预后改善：** 与抗生素和外科技术进步有关

### 优势
- 基于大量病例的综合分析
- 涵盖多种右侧IE类型

### 局限性
- 病例报告为主，可能存在报告偏倚
- 缺乏系统性随访数据
- 无法建立因果关系

---

## 结论

过去5⅓年右侧IE发生了显著变化：患者年龄更轻、赘生物增大，但金黄色葡萄球菌感染仍占主导。并发症谱转变为更多瓣膜关闭不全和栓塞事件，脓肿和穿孔减少。在现代抗菌和外科治疗下，患者预况有所改善。
